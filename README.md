Rock, Paper, Scissors Game
Welcome to the Rock, Paper, Scissors game! This web-based game allows you to play the classic game against the computer. Try your luck and strategy in this simple yet fun game.

Features
Play Rock, Paper, Scissors against a computer AI.
First to three points wins the game.
Responsive design for desktop and mobile play.
Prerequisites
Before you start playing the game, make sure you have a modern web browser such as Chrome, Firefox, or Edge.

Installation
No installation is required! Simply download the files and open the index.html file in your web browser to start playing.

How to Play
Open the index.html file in a web browser.
The game will display three options: Rock, Paper, and Scissors.
Click on your choice.
The computer's choice will be displayed, and the game will announce the round result.
The first to reach three points wins the game.
If a player wins, the option to play again will be displayed.
Game Rules
Rock beats Scissors.
Scissors beats Paper.
Paper beats Rock.
If both players choose the same item, the round is a tie.
Resetting the Game
To play again after a game is finished, click the "Play again?" button.

Files Included
index.html - The main HTML file.
styles.css - CSS file for styling the webpage.
script.js - JavaScript file containing the game logic.
